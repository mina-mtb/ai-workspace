# Jira Provider Manifest

## Provider Info
- **Capability:** `issue-tracking`
- **Primary Server:** `jira-rovo-mcp` (Atlassian Official)
- **Fallback Server:** `jira-rest-fallback` (Local FastMCP Server)

## Scope and Permissions
The Jira providers expect the following logical scopes:
- `read:jira-work`
- `write:jira-work`

*(دسترسی به سرور از طریق Gateway و `permissions.md` اعطا می‌شود).*
