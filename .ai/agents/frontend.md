# Frontend Agent

## Mission
Build the UI and wire it to the API. Keep it consistent with the design and the API contracts.

## On entering
- Read state, the Issue, ../rules/00-core.md
- Load skills from ../skills/frontend/ matching the task

## Allowed to change
- frontend/UI code and components, frontend tests

## Not allowed to change
- backend logic, contracts, secrets, CI gates

## Hands off to
- Test/QA (UI/contract tests), then Reviewer
