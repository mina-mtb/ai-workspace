# Workflow: Start a New Project
1. Fill state/PROJECT.md with the mission + guardrails.
2. Product Owner confirms scope/phase (what's in, what's deferred).
3. Architect drafts initial boundaries + ADR-0001.
4. Orchestrator creates the first batch of Agent Ready Issues.
5. Set up branch protection, CI, and the test command (DevOps).
6. Proceed via add-feature.md per task.
